Пример микросервиса авторизации
-------
_**Чтобы начать работу:**_

---
Для запуска использовать:
```
    make up
```

Для остановки контейнеров использовать:
```
    make down
```

Для запуска миграций:
```
    make migrateup
```

Для отката миграций:
```
    make migratedown
```
Для создания миграции:
```
    make migration name=your_migration_name
```
---
